## File .default.env
```bash
# database
$ HOST=
$ USER_NAME=
$ PASSWORD=
$ DATABASE=

#jwt
$ JWT_KEY=xxxxx
```

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
